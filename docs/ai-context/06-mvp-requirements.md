# Requisitos do MVP

## Funcionais

- conectar no WhatsApp;
- persistir sessão;
- localizar chat da Equatorial;
- iniciar conversa;
- enviar UC;
- confirmar imóvel;
- selecionar referência;
- escolher boleto;
- enviar e-mail;
- baixar PDF;
- salvar PDF.

## Não funcionais

- logs claros;
- timeouts;
- código modular;
- execução local;
- baixa complexidade.

## Critério de sucesso

O MVP será validado quando conseguir:

```txt
Conectar → Conversar → Escolher fatura → Baixar PDF
```
